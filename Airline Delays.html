The goal of this project is to practice essential skills in data science. Tidying and transforming a messy dataset. I am starting with a semi-structured table representing arrival delays for at least two airlines across five different cities.I will be transforming it from a wide format into a long format and filling in missing values to ensure every flight record is correctly attributed to its airline. raw_data <- read_csv(“https://raw.githubusercontent.com/YourName/YourRepo/main/airline_delays.csv”)

Approach

In order to transform the raw flight data into a structured format for analysis, I will be following a three-step process focusing on data integrity, tidying, and comparative analysis.

Challenges

The primary challenge is the missing data in the Airline column; the names “ALASKA” and “AM WEST” only appear on the “on time” rows. I will fix this by using the fill() function from the tidyr package to propagate the airline names downward into the empty cells. This will ensure that every row of data is correctly attributed to the proper airline.

The next challenge is that the original table is wide, with five separate columnms for each city. While readable for humans, this structure prevents easy grouping and calculation. I will use pivot_longer() to collaspe the five city columns into a single columm called City. I will then use pivot_wider() to move the arrival statuses into their own columns. This will result in a Tidy dataset where each row represents a unique observation for one city.

I will calculate the delay rate for each airline globally and then perform a city-by-city comparison. I will create a grouped bar chart to visualize the percentages and they should allow me to investigate the overall delay trends for each individual city.

write.csv(raw_data, "airline_delays.csv", row.names = FALSE)

tidy_flights <- raw_data %>%
  filter(!is.na(status))%>%
  fill(airline) %>%
  pivot_longer(
    cols = 3:7,
    names_to = "city",
    values_to = "count"
  ) %>%
  pivot_wider(
    names_from = status,
    values_from = count
  ) %>%
  rename(on_time = `on time`)
  
  analysis <- tidy_flights %>%
    mutate(
      total = on_time + delayed, 
      pct_delayed = (delayed / total) * 100
      )
      
print(analysis)

# Overall comparison 
overall_summary <- analysis %>%
  group_by(airline) %>%
  summarize(
    total_on_time = sum(on_time),
    total_delayed = sum(delayed),
    overall_pct_delayed = (sum(delayed) / (sum(on_time) + sum(delayed))) * 100)

ggplot(analysis, aes(x = city, y = pct_delayed, fill = airline)) + geom_bar(stat = "identity", position = "dodge") + labs(title = "Delay Percentage by City", y = "Percent Delayed")

Conclusion

While the bar chart shows thhat AM WEST appears to have a lower overall delay rate, a city-by-city breakdown reveals that ALASKA is actually more punctual in every individual location. AM WEST’s overall average is skewed by their high volume of flights in Phoenix, a low-delay city. This is an example of Simpsons Paradox.
